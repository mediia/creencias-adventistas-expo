# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

Please send a message to @maddes, security vulnerabilities will be prioritized above all feature work and resolved asap.
But we're volunteers, coding on our spare time, so cannot commit to a pre-defined timeline.
